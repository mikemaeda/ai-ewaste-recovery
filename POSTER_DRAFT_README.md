# Poster Draft Files

These files continue from the Claude draft and turn the recommended balanced
poster direction into usable project drafts.

## Files Created

| File | Purpose |
| --- | --- |
| `FINAL_POSTER_DRAFT.md` | Final poster-ready text organized by section |
| `poster_draft_48x36.html` | Editable 48 x 36 inch poster layout with figures placed |
| `POSTER_PRESENTATION_SCRIPT.md` | 30-second and 2-minute scripts for presenting |
| `POSTER_ABSTRACT_DRAFTS.md` | Three abstract options plus recommendation |
| `POSTER_CONTENT.md` | Existing content guide and figure inventory |

## How To View The Poster Draft

Open:

```text
poster_draft_48x36.html
```

in a browser.

## How To Export To PDF

1. Open `poster_draft_48x36.html` in Chrome or Edge.
2. Press `Ctrl + P`.
3. Destination: `Save as PDF`.
4. Paper size: custom or poster size if available.
5. Scale: `100%` first. If edges are cut off, use `Fit to printable area`.
6. Save as `poster_draft_48x36.pdf`.

## How To Use With PowerPoint

If your professor requires PowerPoint:

1. Create a blank slide.
2. Set slide size to 48 x 36 inches.
3. Use the layout in `poster_draft_48x36.html` as the visual reference.
4. Copy the text from `FINAL_POSTER_DRAFT.md`.
5. Insert the PNG figures from:

```text
artifacts/reports/poster/
```

Use these core figures no matter what:

- `fig7_classifier_headline.png`
- `fig1_recovery_ranking.png`
- `fig9_material_table.png`
- `fig8_pipeline.png`

## Best Final Version

Use the balanced version from `FINAL_POSTER_DRAFT.md`. It is the strongest
because it presents the project as computer science research while still making
the sustainability value clear.
